# Changelog

All notable changes to the Ndou Ronaldo Soccer Agency website will be documented in this file.


## [1.5.0] - 2025-08-16

### Added
- **Slideshow**: Automatic image cycling implemented on Home page (`index.html`).
- **Accordions**: Interactive expandable sections implemented on Home page (`index.html`) for product features, and on About page (`pages/about.html`) for mission and team information.
- **Current Date/Time Display**: Real-time date and time display implemented on all pages (`index.html`, `pages/about.html`, `pages/contact.html`, `pages/products.html`, `pages/enquire.html`, `pages/testimonials.html`, `pages/news.html`).
- **Newsletter Subscription Form**: Subscription form implemented in the footer on Home page (`index.html`), Testimonials page (`pages/testimonials.html`), and News page (`pages/news.html`).
- **Google Maps Embed**: Interactive Google Maps embed implemented on Contact page (`pages/contact.html`).
- **Contact Form**: Contact form with validation implemented on Contact page (`pages/contact.html`).
- **Search Functionality**: Product search bar implemented on Products page (`pages/products.html`).
- **Price Filter**: Price range dropdown filter implemented on Products page (`pages/products.html`).
- **Product Grid with Modals**: Product display grid with modal details implemented on Products page (`pages/products.html`).
- **Lightbox Gallery**: Image gallery using GLightbox implemented on Products page (`pages/products.html`).
- **Scroll Animations**: Fade-in animations on scroll implemented on Products page (`pages/products.html`).
- **Enquiry Form**: Enquiry form with validation implemented on Enquire page (`pages/enquire.html`).
- **Testimonials Carousel**: Sliding testimonials carousel implemented on Testimonials page (`pages/testimonials.html`).
- **News Articles**: Blog-style news articles implemented on News page (`pages/news.html`).

### Changed
- Updated documentation to reflect current implementations across all pages.

## [1.4.0] - 2025-08-16

### Added
- **Advanced JavaScript Features**: Implemented tabs for product categorization, modals for product details, interactive Leaflet map on contact page, scroll animations, and dynamic content loading
- **Interactive Elements**: Tabs on Products page for Speed, Control, All-Terrain, Professional, Indoor, and Training categories; modals for detailed product views; Leaflet-powered interactive map replacing static Google Maps embed
- **Animations and Effects**: Fade-in animations on scroll for better user experience, hover effects, and smooth transitions
- **Dynamic Content**: Load more products functionality with DOM manipulation to simulate additional content

### Changed
- **Products Page**: Added tabbed interface for better organization, modal triggers on product items, and load more button for dynamic content
- **Contact Page**: Replaced static Google Maps iframe with interactive Leaflet map for enhanced user interaction
- **JavaScript**: Extended script.js with new functions for tabs, modals, animations, dynamic loading, and map initialization
- **CSS Styles**: Added styles for tabs, modals, animations, and load more button
- **README.md**: Updated features section to include advanced JavaScript features and interactive elements

### Fixed
- Enhanced user interaction with more responsive and engaging elements

## [1.2.0] - 2025-08-16

### Added
- **Goals and Objectives Section**: Added a comprehensive "Goals and Objectives" section to `miscellineous/README.md`, detailing clear, detailed, and project-aligned goals that exceed standard requirements, including establishing online presence, product showcasing, customer engagement, mission promotion, accessibility/SEO optimization, and scalability for future growth.

### Changed
- **README.md Structure**: Enhanced the project documentation with a new section outlining strategic objectives, aligning with the agency's mission to provide high-quality soccer footwear and foster business expansion.

## [1.1.0] - 2025-08-16

### Added
- **Products Page** (`pages/products.html`): New page showcasing the agency's soccer boot products
- **Enquire Page** (`pages/enquire.html`): New page with inquiry form for customer inquiries
- **CSS Stylesheet** (`css/styles.css`): Proper external CSS styling for improved visual appeal and responsiveness
- **Assets Directory Updates**: Added new images and media files including logos, product images, and promotional videos

### Changed
- **Styling Approach**: Moved from basic inline styling to external CSS for better maintainability
- **File Structure**: Updated project structure to include dedicated CSS directory and additional pages

### Fixed
- Improved responsive design across devices using proper CSS
